#what is this?

A perfect shadow generator for images

#Installation

`npm i darkShadow --save`

then ...

```
import {darkShadow} from 'darkShadow'

darkShadow({
    shadow_type : 'soft',
    padding: false
})
```

##options

darkShadow supports two arguments

*shadow_type* - _hard/soft_
*padding*-_boolean_
